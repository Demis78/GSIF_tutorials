Product title: Soil property maps of Africa at 1 km
Data set URL: http://www.isric.org/data/soil-property-maps-africa-1-km
Maps produced by: Hengl, T., Heuvelink, G.B.M. and Kempen B. (ISRIC)
Contact: T. Hengl (tom.hengl@wur.nl)
Input data sets: Africa Soil Profile DB (http://www.isric.org/content/africa-soil-profiles-database), WorldGrids 1 km covariates (http://worldgrids.org/doku.php?id=wiki:layers)
Processing steps: https://code.google.com/p/gsif/source/browse/trunk/AFRICA/1km/ 
File naming convention: ******_sd1_M - predicted mean value for the first standard depth (0-5 cm); ******_sd1_L - lower limit of the 90% prediction interval; ******_sd1_U - upper limit of the 90% prediction interval. 
Standard depths: sd1: 0-5 cm, sd2: 5-15 cm, sd3: 15-30 cm, sd4: 30-60 cm, sd5: 60-100 cm, sd6: 100-200 cm. 
Period (temporal coverage approximate): 1950-2005
Spatial resolution (covariates): 1 km
Spatial support for predictions: point support (centre of the grid cell node / voxels)
Coordinate system: +proj=laea +lat_0=5 +lon_0=20 +x_0=0 +y_0=0 +units=m +ellps=WGS84 +datum=WGS84
Total size of the rasters: 7376 x 7877 pixels / 18.3M grid cells (soils with vegetation cover)
Data license (IP policy): http://creativecommons.org/licenses/by-sa/3.0/
